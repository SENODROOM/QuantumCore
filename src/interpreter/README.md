# Quantum Language Interpreter

## Overview

The Quantum Language Interpreter is the core execution engine responsible for evaluating the Abstract Syntax Tree (AST) generated by the Quantum Language compiler. It handles the execution of quantum scripts by interpreting each node in the AST according to its type and content. The interpreter supports various operations including variable declarations, function calls, conditional statements, loops, and more.

## Files and Their Functions

### `execute.cpp`
- **Purpose**: Contains the main entry point for executing the AST.
- **Functions**:
  - `execute(ASTNode* root)`: Initiates the interpretation process from the root of the AST.

### `visit.cpp`
- **Purpose**: Implements the visitor pattern to traverse and interpret different types of AST nodes.
- **Functions**:
  - `visit(ASTNode* node)`: Dispatches the appropriate visit method based on the node's type.

### `execBlock.cpp`
- **Purpose**: Handles the execution of blocks of code, which can contain multiple statements or nested structures.
- **Functions**:
  - `execBlock(BlockNode* block)`: Iteratively executes each statement within the block.

### `execVarDecl.cpp`
- **Purpose**: Interprets variable declaration nodes, setting up variables in the current scope.
- **Functions**:
  - `execVarDecl(VarDeclNode* varDecl)`: Declares and initializes variables as specified by the node.

### `execFunctionDecl.cpp`
- **Purpose**: Handles the declaration and execution of user-defined functions.
- **Functions**:
  - `execFunctionDecl(FunctionDeclNode* funcDecl)`: Defines and registers functions, allowing them to be called later.

### `execClassDecl.cpp`
- **Purpose**: Manages the creation and execution of classes, including their methods and properties.
- **Functions**:
  - `execClassDecl(ClassDeclNode* classDecl)`: Instantiates classes and sets up their member variables and methods.

### `execIf.cpp`
- **Purpose**: Executes conditional statements (`if`, `else`, `elif`).
- **Functions**:
  - `execIf(IfNode* ifNode)`: Evaluates conditions and executes the corresponding branch.

### `execWhile.cpp`
- **Purpose**: Handles while loops, repeatedly executing the loop body until the condition is false.
- **Functions**:
  - `execWhile(WhileNode* whileNode)`: Continuously evaluates the loop condition and executes the body.

### `execFor.cpp`
- **Purpose**: Executes for loops, iterating over sequences or ranges.
- **Functions**:
  - `execFor(ForNode* forNode)`: Iterates through elements and executes the loop body for each element.

### `execReturn.cpp`
- **Purpose**: Manages return statements, transferring control back to the caller and optionally returning a value.
- **Functions**:
  - `execReturn(ReturnNode* returnNode)`: Terminates the current function execution and returns the specified value.

### `execPrint.cpp`
- **Purpose**: Handles print statements, outputting values to the console.
- **Functions**:
  - `execPrint(PrintNode* printNode)`: Prints the expression or value specified by the node.

### `execInput.cpp`
- **Purpose**: Manages input statements, reading data from the user.
- **Functions**:
  - `execInput(InputNode* inputNode)`: Reads input from the user and assigns it to the specified variable.

### `execExprStmt.cpp`
- **Purpose**: Executes expressions that stand alone as statements.
- **Functions**:
  - `execExprStmt(ExprStmtNode* exprStmt)`: Evaluates the expression and discards the result.

### `evaluate.cpp`
- **Purpose**: Provides the core functionality for evaluating expressions.
- **Functions**:
  - `evaluate(ExpressionNode* expr)`: Recursively evaluates expressions and returns the computed value.

### `evalIdentifier.cpp`
- **Purpose**: Handles identifier nodes, resolving variable names to their values.
- **Functions**:
  - `evalIdentifier(IdentifierNode* idNode)`: Retrieves the value associated with the identifier.

### `evalUnary.cpp`
- **Purpose**: Evaluates unary operators such as negation, increment, and decrement.
- **Functions**:
  - `evalUnary(UnaryOpNode* unaryNode)`: Applies the unary operator to its operand.

### `evalAssign.cpp`
- **Purpose**: Handles assignment operations, updating variable values.
- **Functions**:
  - `evalAssign(AssignNode* assignNode)`: Assigns the right-hand side value to the left-hand side variable.

### `callFunction.cpp`
- **Purpose**: Executes function calls, invoking predefined or user-defined functions.
- **Functions**:
  - `callFunction(CallNode* callNode)`: Invokes the function and passes arguments.

### `callNative.cpp`
- **Purpose**: Calls native functions implemented in C++, interfacing between the interpreted language and system-level capabilities.
- **Functions**:
  - `callNative(NativeCallNode* nativeCallNode)`: Invokes native functions and handles their results.

### `callInstanceMethod.cpp`
- **Purpose**: Handles method calls on instances of classes.
- **Functions**:
  - `callInstanceMethod(MethodCallNode* methodCallNode)`: Invokes instance methods and manages object state.

### `evalIndex.cpp`
- **Purpose**: Evaluates indexing operations on arrays and dictionaries.
- **Functions**:
  - `evalIndex(IndexNode* indexNode)`: Retrieves the value at the specified index.

### `evalMember.cpp`
- **Purpose**: Handles attribute access on objects.
- **Functions**:
  - `evalMember(MemberAccessNode* memberNode)`: Accesses attributes of an object.

### `evalArray.cpp`
- **Purpose**: Evaluates array literals and operations
