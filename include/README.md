# QuantumLanguage Compiler - Parser.h

## Overview

The `include/Parser.h` header file is an integral part of the QuantumLanguage compiler, responsible for converting the sequence of tokens generated by the lexer into an Abstract Syntax Tree (AST). The parser follows a top-down approach to construct the AST based on the grammar rules defined for the QuantumLanguage syntax.

## Key Design Decisions

### Error Handling

The parser uses a custom exception class `ParseError` derived from `std::runtime_error`. This decision was made to provide more context about errors during parsing, such as the line and column numbers where the error occurred. This allows for better debugging and user-friendly error messages.

### Pratt Parsing Algorithm

Pratt parsing is employed for handling operator precedence and associativity in expressions. This algorithm is chosen because it provides a flexible and efficient way to parse expressions without requiring complex lookahead mechanisms. It simplifies the implementation of operator precedence parsing and makes it easier to extend the language's syntax.

### Recursive Descent Parser

A recursive descent parser is used instead of a more advanced parser like LL(1) or LR parsers. This choice was made due to its simplicity and ease of understanding. Recursive descent parsers are straightforward to implement and maintain, making them suitable for smaller languages like QuantumLanguage.

## Classes and Functions Documentation

### ParseError Class

**Purpose:** Represents an error encountered during parsing.

**Behaviour:** Inherits from `std::runtime_error` and adds line and column information to provide more context about the error.

**Tradeoffs:** Adds overhead compared to using plain exceptions but provides valuable debugging information.

### Parser Class

**Purpose:** Converts a vector of tokens into an AST.

**Behaviour:** Initializes with a vector of tokens and iteratively parses statements until the end of input is reached.

#### Public Methods

- **explicit Parser(std::vector<Token> tokens):** Constructor initializes the parser with a list of tokens.
- **ASTNodePtr parse():** Parses the entire input stream and returns the root node of the AST.

#### Private Methods

- **Token &current():** Returns the current token being processed.
- **Token &peek(int offset = 1):** Returns the token at the specified offset ahead without advancing the position.
- **Token &consume():** Consumes the current token and advances the position.
- **Token &expect(TokenType t, const std::string &msg):** Ensures the current token matches the expected type, otherwise throws a `ParseError`.
- **bool check(TokenType t) const:** Checks if the next token matches the given type.
- **bool match(TokenType t):** If the next token matches the given type, consumes it and returns true; otherwise, returns false.
- **bool atEnd() const:** Determines if the end of the token stream has been reached.
- **void skipNewlines():** Skips over any newline characters in the token stream.

#### Parsing Methods

- **ASTNodePtr parseStatement():** Parses a single statement.
- **ASTNodePtr parseBlock():** Parses a block of statements enclosed in curly braces `{}`.
- **ASTNodePtr parseBodyOrStatement():** Parses either a block or a single statement, allowing for brace-optional syntax.
- **ASTNodePtr parseVarDecl(bool isConst):** Parses a variable declaration, optionally marked as constant.
- **ASTNodePtr parseFunctionDecl():** Parses a function declaration.
- **ASTNodePtr parseClassDecl():** Parses a class declaration.
- **ASTNodePtr parseIfStmt():** Parses an if statement.
- **ASTNodePtr parseWhileStmt():** Parses a while loop.
- **ASTNodePtr parseForStmt():** Parses a for loop.
- **ASTNodePtr parseReturnStmt():** Parses a return statement.
- **ASTNodePtr parsePrintStmt():** Parses a print statement.
- **ASTNodePtr parseInputStmt():** Parses an input statement.
- **ASTNodePtr parseCoutStmt():** Parses a `cout` statement for outputting multiple expressions.
- **ASTNodePtr parseCinStmt():** Parses a `cin` statement for reading multiple variables.
- **ASTNodePtr parseImportStmt(bool isFrom = false):** Parses an import statement, optionally specifying a module name.
- **ASTNodePtr parseExprStmt():** Parses an expression statement.
- **ASTNodePtr parseCTypeVarDecl(const std::string &typeHint):** Parses a variable declaration with a CType hint, such as `int x = ...` or `int* p = ...`.
- **bool isCTypeKeyword(TokenType t) const:** Checks if the given token type is a CType keyword.

#### Expression Parsing Methods

- **ASTNodePtr parseExpr():** Parses an expression using Pratt parsing.
- **ASTNodePtr parseAssignment():** Parses an assignment expression.
- **ASTNodePtr parseOr():** Parses logical OR expressions.
- **ASTNodePtr parseAnd():** Parses logical AND expressions.
- **ASTNodePtr parseBitwise():** Parses bitwise operators.
- **ASTNodePtr parseEquality():** Parses equality and inequality operators.
- **ASTNodePtr parseComparison():** Parses comparison operators.
- **ASTNodePtr parseShift():** Parses shift operators.
- **ASTNodePtr parseAddSub():** Parses addition and subtraction operators.
- **ASTNodePtr parseMulDiv():** Parses multiplication and division operators.
- **ASTNodePtr parsePower():** Parses power (`**`) operators.
- **ASTNodePtr parseUnary():** Parses unary operators.
- **ASTNodePtr parsePostfix():** Parses postfix operators.
- **ASTNodePtr parsePrimary():** Parses primary expressions.
